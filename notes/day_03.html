1 A Gaussian Process on L2⁢(ℝ)

We will construct a process on L2⁢(ℝ) by taking a limit of a simpler process on the space of sequences ℓ2⁢(ℝ):={(ak)k∈ℤ:∑k∈ℤ|ak|2<∞,ak∈ℝ}. Given a set of independent random variables {Xk}k∈ℤ satisfying Xk∼N⁢(0,1) (think of each Xk as “noise” at the integer k), define Z⁢(a):=∑k∈ℤak⁢Xk for a=(ak)∈ℓ2⁢(ℝ). The collection {Z⁢(a)}a∈ℓ2⁢(ℝ) is a Gaussian process that is centered (i.e., 𝔼⁢[Z⁢(a)]=0) and satisfies

var⁡[Z⁢(a)]=cov⁡[∑k∈ℤak⁢Xk,∑l∈ℤal⁢Xl]=∑k,l∈ℤak⁢al⁢cov⁡[Xk,Xl]=∑k∈ℤak2

for each a∈ℓ2⁢(ℝ). Now recall L2⁢(ℝ):={f:ℝ→ℝ:∫-∞∞(f⁢(x))2⁢𝑑x<∞}. Given f∈L2⁢(ℝ), define a sequence of random variables (Z(N)⁢(f))N∈ℕ by

Z(N)⁢(f):=1N⁢∑k∈ℤf⁢(kN)⁢Xk.

It can be shown that Z(N)⁢(f) converges in distribution to a random variable which we will call Z⁢(f). It can be also shown that the collection {Z⁢(f)}f∈L2⁢(ℝ) is a centered Gaussian process satisfying cov⁡[Z⁢(f),Z⁢(g)]=∫-∞∞f⁢(x)⁢g⁢(x)⁢𝑑x. It turns out that these properties (centered Gaussian and satisfying the above covariance formula) characterize {Z⁢(f)}f∈L2⁢(ℝ), so we might as well define it this way:

Definition 1.1.

The collection {Z⁢(f)}f∈L2⁢(ℝ) is the centered Gaussian process on L2⁢(ℝ) satisfying

cov⁡[Z⁢(f),Z⁢(g)]=∫-∞∞f⁢(x)⁢g⁢(x)⁢𝑑x.

Note that Z⁢(a⁢f+b⁢g)=a⁢Z⁢(f)+b⁢Z⁢(g) for all a,b∈ℝ and all f,g∈L2⁢(ℝ), so f↦Z⁢(f) defines a linear map from L2⁢(ℝ) to the space of random variables. The random variable Z⁢(f) is called the stochastic integral of f, denoted

Z(f)=:∫-∞∞f(t)dWt,

and the symbol d⁢Wt is interpreted as “white noise” that weights the value of f⁢(t). Assuming it is known that cov⁡[d⁢Ws,d⁢Wt]=δs⁢(t)⁢d⁢s⁢d⁢t, where δs is the Dirac δ-function centered at s, we can recover the covariance formula using stochastic integral notation:

cov⁡[Z⁢(f),Z⁢(g)] =cov⁡[∫-∞∞f⁢(s)⁢𝑑Ws,∫-∞∞g⁢(t)⁢𝑑Wt]
=∫-∞∞∫-∞∞f⁢(s)⁢g⁢(t)⁢cov⁡[d⁢Ws,d⁢Wt]=∫-∞∞f⁢(t)⁢g⁢(t)⁢𝑑t.
